# wow-macros-interface-and-tweaks

**Built with:** Node.js/JavaScript

## Getting Started

Clone the repository:

```bash
git clone https://github.com/suhteevah/wow-macros-interface-and-tweaks.git
cd wow-macros-interface-and-tweaks
```

## Installation

```bash
npm install
npm start
```
